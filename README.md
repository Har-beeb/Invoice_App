# Invoice Management App

A fully responsive, pixel-perfect invoice management dashboard built with React and modern Tailwind CSS v4.

## 🚀 Features

- **Responsive Design:** Mobile-first architecture that seamlessly scales from mobile devices to large desktop screens.
- **Theme Toggle:** Fully functional Dark and Light modes that remember your preference.
- **Custom Filtering:** A custom-built, multi-select checkbox dropdown to filter invoices by status (Draft, Pending, Paid).
- **Pixel-Perfect UI:** Built entirely with scalable SVGs (no text-based icons) and customized League Spartan typography to match strict professional design specs.
- **Interactive UI:** Smooth hover states, snappy button animations, and custom UI components replacing default browser elements.
- **Empty States:** Beautifully handled empty states and fallback logic when no invoices match the selected filters.

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS v4
- **Typography:** Google Fonts (League Spartan)
- **Icons:** Inline SVGs

## 💻 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository
   `git clone https://github.com/your_username_/Project-Name.git`

2. Navigate to the project directory
   `cd invoice-app`

3. Install NPM packages
   `npm install`

4. Start the development server
   `npm run dev`
